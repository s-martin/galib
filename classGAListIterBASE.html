Detailed Description

This is the base class for iterators for the list objects.

We define this class separately from the List object so that you can have multiple interators for each list and so that you can more easily customize the traversal algorithms within the iterator. From the object point of view, the way you traverse a list is independent of how you represent the list. Like the ListBASE object, this object doesn't do any memory allocation or deallocation. All we do is provide list traversal. Notice that we keep a 'current location' in the list - whatever your last query was is stored as the node, so if you refer to the current member, you'll get your last query. If you pass a NULL node to these routines they will not break; passing a NULL will result in a no-op, and NULL will be returned.
